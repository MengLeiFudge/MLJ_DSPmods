# 建筑分析

- 建筑特性通过消耗芯片+对应分馏塔解锁。特性分为10层，顺序如下：
    - Lv1：增产优化I
        - 分馏带有增产点数的物品时，增产剂效果提升（+0% -> +10%）
    - Lv2：电力消耗I
        - 分馏塔的待机能耗、运行能耗降低（-0% -> -10%）
    - Lv3：搬运强化
        - 分馏塔的流动输出将尽可能集装后再输出；产物到达上限时，物品仍可通过分馏塔
    - Lv4：增产优化II
        - 分馏带有增产点数的物品时，增产剂效果提升（+10% -> +30%）
    - Lv5：电力消耗II
        - 分馏塔的待机能耗、运行能耗降低（-10% -> -20%）
    - Lv6：特殊特质1
        - 不同分馏塔的特质不同
    - Lv7：增产优化III
        - 分馏带有增产点数的物品时，增产剂效果提升（+30% -> +60%）
    - Lv8：电力消耗III
        - 分馏塔的待机能耗、运行能耗降低（-20% -> -35%）
    - Lv9：输出集装
        - 分馏塔的产物输出将尽可能集装后再输出
    - Lv10：增产优化IV
        - 分馏带有增产点数的物品时，增产剂效果提升（+60% -> +100%）
    - Lv11：电力消耗IV
        - 分馏塔的待机能耗、运行能耗降低（-35% -> -50%）
    - Lv12：特殊特质2
        - 不同分馏塔的特质不同

## 交互塔

- 功能
    - 将分馏塔原胚培养为指定的分馏塔
    - 将分馏塔还原为更多的、随机的分馏塔原胚
    - 上传物品到数据中心（速率无限制，相当于垃圾桶）
- 特性
    - 分馏塔献祭
        - 数据中心中，超过100的分馏塔会自动分解，并为同类型的分馏塔提供限时全局buff。
        - 每个被分解的塔都会为同类型分馏塔提供60s的1%速度增幅。
    - 献祭永驻
        - 被献祭的分馏塔，每到2的次方倍数，就会额外给予永久的10%速率增幅。

## 矿物复制塔

- 功能
    - 将原矿复制为2份
    - 所有配方无损毁
- 特性
    - 增产点数附加
        - 当流动输入充足时，分馏塔会粉碎一部分原料，并为产物输出附加至多10点增产点数。
    - 内部循环
        - 流动输出、产物输出会自动回填至输入，无需外部供料即可输出。

## 点数聚集塔

- 功能
    - 将输入物品携带的增产点数集中到小部分物品上，达到至多10点。
    - 所有配方无损毁
- 特性
    - 自动喷涂
        - 自动使用分馏数据中心的增产剂对原料进行喷涂。
    - 点数拆分
        - 使用优化的点数移动方式，原料的1点增产点数转移后可为产物附加2点增产点数。

## 转化塔

- 功能
    - 将输入的物品转化为与其相关的其他物品。
- 特性
    - 原子重组
        - 原料的增产点数会无损转移到产物上。
    - 笼中窥梦
        - 20%概率产出两份产物。

# 符文系统

抽奖杂项奖励、分馏塔副产物、分解不需要的符文，都可以得到精华。精华可用于符文升级。
奖券抽奖可以得到符文。符文有五个星级，每个符文可以携带多个词条。词条分为主词条和副词条。

词条奖池：（作为主词条时，一级提升的程度）（最终效果=原始值*(1+增幅百分比)）
速度词条：速度加成 +50% 电力消耗 +70%
产能词条：产能加成 +10% 电力消耗 +80% 速度加成 -15%
节能词条：电力消耗 -50% （电力消耗最低为-80%）
增产词条：增产剂效果 +5%

主词条：
主词条就是四种词条之一。
每升1级，主词条的效果就会叠加一次。无论什么星级，主词条每级的提升都是一样的，但高星级有更高的等级上限。

副词条：
副词条在词条奖池中抽取，可能抽到任何一个词条。
每升4级（也就是等级提升到4/8/12/16/20的时候），就会随机抽取一个副词条叠加在符文上。
对于n星符文，单次副词条范围是0.7*0.5n至1.0*0.5n。
前者在0.7、0.8、0.9、1.0四个数值中随机挑选一个（都是25%概率），后者是星级越高加成越高，最高相当于2.5级的提升。

符文插在符文槽中，初始符文槽数目为0。前五个矩阵解锁时，会使符文槽位+1。
最终有5个槽位，也就是一共可以插5个符文。可以自由搭配想要的符文。符文对全局生效。

符文抽取：
符文只能通过“符文奖池”抽取。奖池可以获得1-5星的符文，还有不同的精华。奖券不能使用黑雾奖券。
抽到符文的概率固定为10%，1-5星按照 16:8:4:2:1 的比例（如果对应星级可以被抽到的话）。
电磁奖券只能抽到1星，所以是10%*(16/(16))概率1星，其余精华；
能量奖券可以抽到1星和2星，也就是10%*(16/(16+8))概率1星，10%*(8/(16+8))概率2星，其余精华。
引力奖券至多可以抽到5星，规则与上面一样。
对于宇宙奖券，虽然也是只能抽到5星，但是词条数目必定是上限（例如5星必定是4词条，四星必定是3词条）。
