词条奖池：（作为主词条时，一级提升的程度）（最终效果=原始值*(1+增幅百分比)）
速度词条：速度加成 +50% 电力消耗 +70%

主词条：
主词条就是四种词条之一。
每升1级，主词条的效果就会叠加一次。无论什么星级，主词条每级的提升都是一样的，但高星级有更高的等级上限。

副词条：
副词条在词条奖池中抽取，可能抽到任何一个词条。
每升4级（也就是等级提升到4/8/12/16/20的时候），就会随机抽取一个副词条叠加在符文上。
对于n星符文，单次副词条范围是0.7*0.5n至1.0*0.5n。
前者在0.7、0.8、0.9、1.0四个数值中随机挑选一个（都是25%概率），后者是星级越高加成越高，最高相当于2.5级的提升。

符文插在符文槽中，初始符文槽数目为0。前五个矩阵解锁时，会使符文槽位+1。
最终有5个槽位，也就是一共可以插5个符文。可以自由搭配想要的符文。符文对全局生效。

符文抽取：
符文只能通过“符文奖池”抽取。奖池可以获得1-5星的符文，还有不同的精华。奖券不能使用黑雾奖券。
抽到符文的概率固定为10%，1-5星按照 16:8:4:2:1 的比例（如果对应星级可以被抽到的话）。
电磁奖券只能抽到1星，所以是10%*(16/(16))概率1星，其余精华；
能量奖券可以抽到1星和2星，也就是10%*(16/(16+8))概率1星，10%*(8/(16+8))概率2星，其余精华。
引力奖券至多可以抽到5星，规则与上面一样。
对于宇宙奖券，虽然也是只能抽到5星，但是词条数目必定是上限（例如5星必定是4词条，四星必定是3词条）。

## 二、核心问题诊断

### 🔴 问题1：抽卡缺乏保底机制（最致命）

**现状**：当前抽奖系统有"谢谢惠顾"填充空奖，但**没有任何保底计数器**。

**解决方案**：引入双层保底机制
- **软保底**：连续N次未出目标稀有度时，概率逐渐上升
- **硬保底**：绝对保底，必出目标（配方池第50抽必出未解锁配方）
- **自动化联动**：工厂每生产X价值物品，额外获得1次保底计数

### 🔴 问题2：工厂与抽卡的反馈回路断裂

**现状**：两个系统基本是各玩各的。

**解决方案**：
1. **分馏副产物 → 残片**：分馏塔运作时产出"残片"，可兑换任意奖券
2. **配方图鉴进度 → 全局加成**：解锁50%某类配方 → 该类配方+5%成功率
3. **符文套装效果 → 工厂特殊能力**：4件套解锁"超频共鸣"等

### 🟡 问题3：进度系统完全缺失

**现状**：MainTask、RecurringTask、Achievements 全是空壳。

**解决方案**：
- 主线任务：引导链（序章→第一章→...→终章）
- 循环任务：生产型/抽卡型/升级型/探索型/挑战型
- 成就系统：铜/银/金/铂金 → 永久加成

### 🟡 问题4：符文系统缺乏构建深度

**现状**：没有套装效果，符文退化为纯数值堆叠。

**解决方案**（TODO.md已设计）：
- 2件套：额外提升主词条效果10%
- 4件套：解锁特殊机制（超频共鸣、物质守恒等）

### 🟡 问题5：缺乏终局循环（End-Game Loop）

**现状**：配方最高Level 10，建筑最高+20。到达顶端后无事可做。

**解决方案**：**退火/重置系统**
```
满级配方 ──→ 消耗宇宙矩阵"退火" ──→ 配方等级重置为0
                                      ↓
                              获得"回响"永久加成
                              （该配方永久+X%效率）
                                      ↓
                              重新升级配方（更快）
                                      ↓
                              再次满级 → 再次退火...
```

## 三、战略优化方案 —— 七大突破方向

### 突破一：保底系统
| 层级 | 机制 | 触发条件 |
|------|------|---------|
| 软保底 | 概率逐渐上升 | 配方池：第30抽起+5%/抽 |
| 硬保底 | 必出目标 | 配方池：第50抽必出未解锁配方 |
| 命定保底 | 必出特定类型 | 连续2次未出指定类型后，第3次必出 |

### 突破二：工厂-抽卡螺旋上升系统
```
工厂产出 ──→ 奖券/残片 ──→ 抽卡
    ↑                         │
    │    ┌──────────────────┘
    │    ↓
    │  新配方解锁 ──→ 新分馏产线
    │  符文加成    ──→ 全局效率提升
    │  成就奖励    ──→ 永久加成
    └───── 工厂产出更多 ←──────────────
```

### 突破三：任务/成就系统
- **主线任务**：序章→第一章→...→终章（万物归一）
- **循环任务**：生产型/抽卡型/升级型/探索型/挑战型
- **成就系统**：铜(+1%)→银(+2%)→金(+5%)→铂金(+10%)

### 突破四：Banner轮换与限定机制
| 常驻池 | 永远可用 | 基础配方、基础符文 |
| UP池 | 每48小时轮换 | 特定类型配方UP概率翻倍 |
| 精选池 | 每周轮换 | 限定高星符文主词条类型 |
| 挑战池 | 需完成条件解锁 | 高价值奖励但需要"挑战门票" |

### 突破五：退火/重置机制
- 配方等级重置为0
- 获得1点"回响"点数（永久+2%成功率/点）
- 升级速度提升20%（越刷越快）
- 终极目标：全配方满回响 → 解锁"超越模式"

## 五、设计哲学总结

> **"分馏应该提供便利，但是不能跳过游戏进程。"** —— 开发日记 v1.4.1

> **"我想要的，是左脚踩右脚上天的机会。"** —— TODO.md

1. 🏭 **工厂是引擎**：每一座分馏塔都在产出抽卡资源

3. 🔄 **退火是涡轮**：满级后重置获得永久加成，无限循环
