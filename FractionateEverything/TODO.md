# 符文系统

抽奖杂项奖励、分馏塔副产物、分解不需要的符文，都可以得到精华。精华可用于符文升级。
奖券抽奖可以得到符文。符文有五个星级，每个符文可以携带多个词条。词条分为主词条和副词条。

（表格后两项基准值是主词条一级的提升量）
星级        最高等级  副词条初始数量 副词条抽取总数      副词条单次强化步进     主词条满级数值  理论上限
五星 (金色)   20级       2-3条           5次       1.75 / 2.00 / 2.25 / 2.50       20         32.5
四星 (紫色)   16级         2条           4次       1.40 / 1.60 / 1.80 / 2.00       16         24.0
三星 (蓝色)   12级       1-2条           3次       1.05 / 1.20 / 1.35 / 1.50       12         16.5
二星 (绿色)    8级         1条           2次       0.70 / 0.80 / 0.90 / 1.00        8         10.0
一星 (白色)    4级         0条           1次       0.35 / 0.40 / 0.45 / 0.50        4          4.5

词条奖池：（作为主词条时，一级提升的程度）（最终效果=原始值*(1+增幅百分比)）
速度词条：速度加成 +50%   电力消耗 +70%
产能词条：产能加成 +10%   电力消耗 +80%   速度加成 -15%
节能词条：电力消耗 -50%   （电力消耗最低为-80%）
增产词条：增产剂效果 +5%

主词条：
主词条就是四种词条之一。
每升1级，主词条的效果就会叠加一次。无论什么星级，主词条每级的提升都是一样的，但高星级有更高的等级上限。

副词条：
副词条在词条奖池中抽取，可能抽到任何一个词条。
每升4级（也就是等级提升到4/8/12/16/20的时候），就会随机抽取一个副词条叠加在符文上。
对于n星符文，单次副词条范围是0.7*0.5n至1.0*0.5n。
前者在0.7、0.8、0.9、1.0四个数值中随机挑选一个（都是25%概率），后者是星级越高加成越高，最高相当于2.5级的提升。

符文插在符文槽中，初始符文槽数目为0。前五个矩阵解锁时，会使符文槽位+1。
最终有5个槽位，也就是一共可以插5个符文。可以自由搭配想要的符文。符文对全局生效。
