# UI-抽奖

- 添加自选卡池
    + 有限制次数，每种奖券分别计算，具体限制多少次未定
    + 选择10项内容。其中2项固定为核心、芯片，其余8项为自选配方；配方选择范围是当前奖券对应科技层次的任何配方之一
    + 要么提升概率但奖券需求不变，要么大幅提升概率且奖券需求大幅提升
    + 说实话，我不确定“自选卡池+随机商店”这个设计是否真的能让玩家获取到想要的配方，而且还不能过渡泛滥

- 配方池的配方分为两种，未解锁的按照0.6%、90保底处理，已解锁的提升到6%
    + 说到底，还是配方太多，原神模式抽奖出金概率太低。这个至今没想好要怎么平衡，需要重点讨论一下
    + 要不然，配方抽取按照“物品”的概念获取，也就是抽到这个物品，则这个物品的所有配方都获取
- 添加卡池提示，防止选错，重点在于做出差异化；其实我觉得将概率直接显示出来就有足够的差异了
