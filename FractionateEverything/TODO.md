# 要做的改动

## 一、当前契合新系统的大改动项

1. 抽卡整体重构
   - 资源来源改为“当前阶段矩阵 + 残片”
   - 奖池结构改为：开线池 / 原胚闭环池 / 成长池 / 流派聚焦层
   - 常规模式与速通模式使用不同抽奖体系
   - 抽卡类型自选暂不进入 2.3 首版，后续再评估是否追加

2. 配方成长重构
   - `BaseRecipe` 增加语义标记
   - 生产型 / 工具型 / 特殊成长型配方分流处理
   - 取消“所有分馏配方统一 11 次完全体”逻辑

3. 建筑成长重构
   - 坚持塔种全局成长
   - 普通等级靠塔种经验
   - 关键节点使用残片 + 当前阶段矩阵突破

4. 原版配方增强
   - 改为“落后一层”的阶段性优化奖励
   - 只有下一层科技全部研究完成后才开放低一层增强

5. 任务 / 成就 / 图鉴重构
   - 主线任务重写为新主线 10 阶段
   - 循环任务重写为 5 类稳定补充任务
   - 图鉴加成全部迁入成就系统
   - 成就系统成为全局增幅唯一入口

6. 速通模式
   - 采用阶段箱池 + 简化原胚池 + 简化成长池
   - 突出肉鸽式押一路成长

7. 黑雾支线整合
   - 黑雾不作为主线矩阵层并列池
   - 作为从信息/引力开始插入的战斗支线内容

## 五、抽卡整体重构执行计划

### 已固化原则

- 抽卡的快乐核心是：
  “抽到的东西值得玩家回去重搭一条线”，而不是单次爆大奖。
- 奖池分类优先按“玩家目标”设计，而不是按奖励外观设计。
- 原胚池保留为常驻普通闭环池，强调原胚循环与献祭正反馈。
- VIP 移除，不再参与抽卡与商店数值。
- 升级少决策，流派多决策。
- 建筑成长采用塔种全局成长，而不是单建筑个体成长。
- 常规模式与速通模式共享世界观，但使用不同抽奖体系。

### 模组内五个核心目标

1. 开线
   解锁新的分馏配方、新链路、新生产分支。

2. 杠杆
   解锁新的塔种、关键工具入口与跨时代跳板。

3. 闭环
   形成原胚循环、献祭、精馏经济等自增殖或自供给正反馈。

4. 压缩
   通过更少物流、更低电耗、更强效果完成同等产能。

5. 构筑 / 流派
   让玩家围绕某一方向长期投入并获得明显阶段进展。

### 流派与塔种归属

- 复制扩张流
  核心塔：矿物复制塔
  边界：只允许复制矿物，不再继续外扩复制边界。

- 转化跃迁流
  核心塔：转化塔
  重点：跨时代工具、阶段跳板、科技层次压缩。
  风险：必须重做建筑类转化配方的成长逻辑。

- 交互物流流
  核心塔：交互塔、行星内物流交互站、星际物流交互站
  重点：上传/下载、库存便利、原胚循环、献祭闭环。

- 精馏经济流
  核心塔：精馏塔
  重点：抽卡副资源、终局压缩或两段式经济转换。

- 点数辅助流
  核心塔：点数聚集塔
  定位：通用支援位，不单独承担主流派池。
  重点：增产剂效果放大、全局工艺优化。


### 原版配方增强

原版配方增强不再作为终局鸡肋内容，而改为“落后一层的阶段性优化奖励”。

规则：

- 对每个原版科技，按“研究该科技所需的最高层次矩阵”划分阶段。
- 只有在更高一层科技全部研究完成后，才开放低一层原版配方增强。

示例：

- 完成全部能量矩阵层科技 -> 开放电磁矩阵层原版配方增强
- 完成全部结构矩阵层科技 -> 开放能量矩阵层原版配方增强
- 完成全部信息矩阵层科技 -> 开放结构矩阵层原版配方增强
- 完成全部引力矩阵层科技 -> 开放信息矩阵层原版配方增强
- 完成终局通关科技 -> 开放引力矩阵层原版配方增强

### 配方类型重构

`BaseRecipe` 保留，但后续需要增加语义标记，至少区分：

- 生产型配方
  持续消耗、长期构成产线，可保留成长曲线。

- 工具 / 解锁型配方
  例如建筑偷跑、阶段工具，不适合高重复抽取强化。

- 特殊成长型配方
  例如带有明确终局优化、研究点推进、特殊机制的配方。

目标：

- 不再让所有分馏配方共用同一种“重复抽取 11 次完全体”逻辑。
- 重复抽取更多转为研究点、残片、回响、定向材料，而不是继续简单堆等级。

### 建筑成长重构

建筑成长继续采用“塔种全局成长”：

- 某种塔从 +0 -> +1
- 等于该塔种所有实例整体升级

成长结构：

1. 常规成长
    - 分馏塔：塔种总处理量
    - 交互站：上传/下载消耗总能量
    - 原胚循环：循环次数、献祭总量、共鸣累计量

2. 关键节点
    - 少量关键等级（如 3 / 6 / 9 / 12）
    - 使用关键耗材突破
    - 解锁机制变化或特性

3. 资源结构
   - 普通等级：使用塔种经验或更通用副资源
   - 关键节点：使用残片 + 当前阶段矩阵

### 全局增幅重构

所有全局增幅统一只通过成就系统发放。

包括：

- 图鉴收集度节点增幅
- 工厂 / 物流 / 发电 / 分馏成功率等全局增幅
- 未来的全星区增幅

实现原则：

- 不再让图鉴、任务、独立系统分别直接挂被动 buff
- 统一改成“达成某一里程碑 -> 解锁对应成就 -> 成就系统汇总最终增幅”
- 成就系统成为全局增幅的唯一展示与汇总入口

### 关于流派专属道具的当前结论

当前版本的优先原则是“简化”，因此：

- 暂不把“每个流派都有一种专属稀有道具”作为 2.3 版本的必做项
- 若未来确认成长深度不足，再考虑补充

当前推荐方案：

- 不再依赖通用增幅芯片
- 也暂不新增多种流派专属道具
- 建筑关键节点直接使用：
  - 残片
  - 当前阶段矩阵

理由：

- 若只是为了拆币种而拆，会显著增加理解成本
- 当前模组内容已经很多，应优先压缩资源种类
- “升级少决策，流派多决策”意味着决策重点应放在抽什么、追什么，而不是管理一堆道具

### 符文系统当前结论

当前符文系统与 2.3 重构主架构不契合，建议直接退出本轮核心方案。

主要原因：

- 当前符文是“全星区可装卸的全局装备”，影响范围过大
- RuneManager 的总加成主要只在 RuneMenu 中展示，未真正接入新的抽卡/成长主循环
- 所谓“精华”当前全部映射到残片，资源语义并不独立
- 若改成机器插件，又会显著增加管理复杂度，并失去原有收藏成长感

2.3 的处理建议：

- 不把符文系统纳入抽卡重构主线
- 主线任务不再围绕符文设计
- 循环任务不再保留符文类别
- 成就系统中的符文成就整组移除或替换
- 符文 UI 可以先保留为废弃旧系统，后续若要重做，另起一套方案

### 阶段跃迁

阶段跃迁的定义：

不是直接给一堆资源，而是允许玩家在关键节点提前拿到一个跨时代杠杆。

典型候选：

- 采矿机 -> 大型采矿机
- 化工厂 -> 量子化工厂
- 电弧熔炉 -> 位面熔炉
- 制造台 MkI/II/III -> 重组式制造台
- 矩阵研究站 -> 自演化研究站
- 行星内物流运输站 -> 星际物流运输站 -> 轨道采集器

发电相关处理原则：

- 不直接把发电建筑本身作为奖池核心奖励
- 更适合做：
    - 当前发电阶段全局增幅
    - 或任务/成就驱动的全星区增幅
- 尽量避免为发电再额外开新奖池或独立新系统

### 模式

保留两种主模式：

1. 常规模式
   面向长期发育、探索、搭线、逐步解锁内容的玩家。

2. 速通模式
   面向希望快速打通、快速变强、把模组当作短局体验的玩家。

严格模式当前不再作为必须主模式。

### 常规模式抽奖体系

奖池 / 系统结构：

- 开线池
  新分馏配方、回响、阶段性原版配方增强。

- 原胚闭环池
  原胚、定向原胚、闭环与献祭相关奖励。

- 成长池
  定向补差、关键成长材料、原版配方增强入口。

- 流派聚焦层
  不是独立奖池，而是覆盖在上述系统之上的“聚焦方向”。
  聚焦后会影响：
  - 开线池中的相关配方/回响权重
  - 原胚闭环池中的相关原胚/定向原胚权重
  - 成长池中的对应定向兑换项与成长项

资源来源：

- 引导层
  主线、成就、首次解锁、阶段礼包。

- 稳定层
  当前阶段矩阵的自动化生产，以及精馏塔对矩阵的精馏副产物。

- 定向层
  残片与当前阶段矩阵直耗。

轮换节奏：

- 基础池常驻
- 流派聚焦以“阶段推进 + 手动切换/刷新”为主
- 不采用纯时间强制轮换

额外原则：

- 2.3 版本建议移除实体奖券，直接以矩阵和残片承担抽卡消耗。
- 回响与建筑经验作为系统内部进度，不再额外做成独立流通货币。

进一步固化：

- `BuildingTrainRecipe` 不进入开线池
  原胚培养属于基础闭环基础设施，应由原胚池与对应科技承担

- `PointAggregateRecipe` 不进入开线池
  点数聚集配方当前是默认满级，不存在抽取必要

- `RectificationRecipe` 不进入开线池
  精馏塔若固定为“矩阵 -> 残片”的稳定副资源塔，则对应配方应自动开放或由建筑科技承担

- 开线池核心内容实际聚焦于：
  - `MineralCopyRecipe`
  - `ConversionRecipe`
  - 以及未来新增的真正“开新线”的分馏配方类型

### 速通模式抽奖体系

奖池 / 系统结构：

- 阶段箱池
  固定奖品，每抽必得一个阶段目标。

- 流派聚焦层（简化）
  不做复杂切换。
  可以采用：
  - 每阶段从 3 个主题里选 1 个
  - 或局内随机给 1 个主题
  若提供多选一：
  - 选中的主题后续出现概率提高，且效果更强
  - 未选中的主题后续出现概率降低，且效果弱化
  以突出肉鸽式“押一路成长到极致”的体验
  主题直接影响阶段箱池和原胚池的奖励倾向。

- 简化原胚池
  保留原胚闭环，但掉率和成型速度更激进。

- 简化成长池
  只保留关键突破与少量定向补差。

资源来源：

- 任务 / 阶段奖励
- 矩阵直耗
- 适量残片

轮换节奏：

- 可接受更快节奏
- 允许短周期主题轮换或局内随机主题

### 当前采用的资源来源方向

#### 常规模式

采用“矩阵直耗 + 残片副资源”的混合结构：

- 当前阶段矩阵
  直接用于开线池 / 阶段跃迁 / 部分关键节点突破

- 残片
  主要通过精馏塔精馏矩阵、副产物、奖励等方式获得
  用于原胚闭环池、流派聚焦后的加权抽取、成长补差

- 回响 / 建筑经验
  作为系统内部进度，不额外做成独立货币

#### 速通模式

采用“任务/阶段奖励 + 矩阵直耗”为主的简化结构：

- 大量阶段奖励直接给抽取机会
- 矩阵仍可直接用于阶段箱池
- 残片存在，但不承担主要供给责任
- 尽量不额外引入慢速积累货币

### 当前执行顺序

1. 先决定常规模式的抽卡资源来源细则
2. 再固化速通模式的简化资源来源
3. 再落实奖池顶层分类与各池职责
4. 再重构配方成长与建筑成长
5. 最后重写任务、教程、商店和 UI 文案

### 当前待实现细节

- 精馏塔精馏矩阵时，残片产出倍率如何按矩阵阶段分层？

### 下一步草案：常规模式资源细化

#### 常规模式仅保留两种外部货币

1. 当前阶段矩阵
2. 残片

不保留：

- 实体奖券
- 流派专属稀有道具（2.3 暂不采用）
- 通用增幅芯片作为核心成长货币

#### 常规模式资源用途

- 当前阶段矩阵
  - 开线池抽取
  - 阶段跃迁
  - 建筑关键节点突破

- 残片
  - 原胚闭环池抽取
  - 成长池补差
  - 流派聚焦后的额外抽取与定向投入

#### 常规模式资源来源

1. 引导层
   - 主线任务
   - 成就
   - 首次解锁
   - 阶段礼包

2. 稳定层
   - 当前阶段矩阵的自动化生产
   - 精馏塔精馏矩阵获得残片

3. 定向层
   - 残片兑换
   - 当前阶段矩阵直耗

#### 常规模式系统分工

- 开线池
  负责新分馏配方、回响、阶段性原版配方增强

- 原胚闭环池
  负责原胚、定向原胚、闭环奖励

- 成长池
  负责建筑关键节点、原版配方增强、定向补差

- 流派聚焦层
  不是独立奖池，而是对以上三者做方向加权

### 下一步草案：速通模式资源细化

#### 速通模式原则

- 快速获得强力奖励
- 减少慢速积累货币
- 放大肉鸽式押一路成长

#### 速通模式资源

- 当前阶段矩阵
- 少量残片
- 阶段奖励直接给抽取机会

#### 速通模式系统分工

- 阶段箱池
  每抽必得阶段目标

- 原胚池
  掉率更激进，尽快形成原胚闭环

- 简化成长池
  只保留关键突破与少量补差

- 流派聚焦层（简化）
  - 每阶段三选一或局内随机一个主题
  - 选中的主题后续出现概率更高、效果更强
  - 未选中的主题后续出现概率更低、效果更弱

### 下一步草案：开线池阶段切分

#### 开线池的顶层原则

- 开线池仍然只有一个顶层入口，不拆成多个矩阵独立奖池。
- 但内部内容按“当前最高已解锁矩阵层次”精确分层。
- 当前阶段矩阵只负责当前阶段的开线内容，不回头承担低阶段主内容。

#### 开线池内部结构

每个矩阵层次内部固定分成两个区块：

1. 本层新分馏配方
   - 当前阶段新增的生产型分馏配方
   - 当前阶段必要的工具/解锁型分馏配方

2. 低一层原版配方增强
   - 只放“落后一层”的原版配方增强项
   - 不再作为单独终局内容

抽到重复内容时：

- 分馏配方 -> 回响
- 原版增强 -> 对应增强进度或强化层数推进

#### 开线池的阶段切分

开线池内部按 6 层矩阵精确推进：

1. 电磁矩阵层
2. 能量矩阵层
3. 结构矩阵层
4. 信息矩阵层
5. 引力矩阵层
6. 宇宙矩阵层

但对玩家的总体阶段感，仍按更大的 4 段理解：

- 前期：电磁 + 能量
- 中期：结构 + 信息
- 后期：引力
- 终局：宇宙

黑雾不作为“终局主线矩阵层”的简单并列层处理。
黑雾更适合定义为：

- 独立于主产线外的战斗支线
- 从信息矩阵 / 引力矩阵阶段开始逐步插入
- 使用黑雾矩阵与黑雾独有掉落，解锁隐藏科技与隐藏建筑

因此：

- 主线开线池仍按 6 层矩阵推进
- 黑雾开线内容作为“支线子层”挂在信息 / 引力 / 宇宙阶段之后逐步开放

这样做的理由：

- 内容投放精度足够细
- UI 不会因为开太多顶层池而碎裂
- 阶段感仍然清晰

#### 开线池的内容边界

开线池不负责：

- 原胚
- 建筑成长材料
- 发电全局增幅
- 通用资源补给

开线池只负责：

- 新分馏配方
- 回响
- 原版配方增强

#### 开线池与阶段跃迁的关系

阶段跃迁不直接放在开线池里。

例如：

- 大型采矿机
- 位面熔炉
- 量子化工厂
- 重组式制造台
- 自演化研究站

这类“跨时代跳板”更适合放在成长池 / 阶段跃迁入口里处理，
而不是和分馏配方混在同一个随机池中。

#### 开线池阶段内容草案

##### 电磁矩阵层

- 目标：
  建立最基础的分馏玩法认知
- 主要内容：
  - 矿物复制：
    木材、植物燃料、铁矿、铜矿、石矿、煤矿、水等基础资源
  - 转化：
    基础输送、基础分拣、基础制造、基础冶炼、基础储存、基础供电
  - 早期配方回响
- 不包含：
  - 原胚
  - 建筑成长
  - 原版配方增强

##### 能量矩阵层

- 目标：
  让玩家开始接触油、燃料、化工与增产相关链路
- 主要内容：
  - 矿物复制：
    原油、硫酸、氢、重氢、临界光子等液体/气体资源
  - 转化：
    油化工、燃料棒、增产剂起步、基础流体设施与其前后链路
  - 电磁矩阵层的原版配方增强
  - 更完整的前期转化配方

##### 结构矩阵层

- 目标：
  进入真正的中期工业化阶段
- 主要内容：
  - 矿物复制：
    钛石、可燃冰、金伯利矿石、有机晶体等中期矿物与稀有资源
  - 转化：
    钛、合金、高强度材料、储液/仓储升级、中期工业建筑链
  - 能量矩阵层的原版配方增强
  - 与大矿机、位面熔炉前置相关的配方开线内容

##### 信息矩阵层

- 目标：
  打开半导体、高级物流、粒子链和中后期工艺压缩
- 主要内容：
  - 矿物复制：
    分形硅石、光栅石、刺笋结晶等半导体与高阶材料相关稀有资源
  - 转化：
    半导体链、粒子链、高级物流、制造台 MkIII、量子化工厂、自演化研究站前置
  - 结构矩阵层的原版配方增强
  - 与制造台 MkIII、量子化工厂、自演化研究站前置相关的开线内容
- 同阶段附加：
  - 黑雾支线第一批开线内容开始出现

##### 引力矩阵层

- 目标：
  打开跨星际跃迁、后期 build 形成与主流派分化
- 主要内容：
  - 矿物复制：
    单极磁石、反物质及其他后期高价值资源
  - 转化：
    引力透镜、空间翘曲、轨道采集器、人造恒星、戴森球后段相关前置
  - 信息矩阵层的原版配方增强
  - 与轨道采集器、人造恒星、戴森球后段相关的开线内容
- 同阶段附加：
  - 黑雾支线第二批开线内容继续推进

##### 宇宙矩阵层

- 目标：
  承接终局和长期复利内容，而不是继续大量投放基础开线内容
- 主要内容：
  - 矿物复制：
    宇宙终局阶段仍需持续消耗的超高阶资源
  - 转化：
    终局 build、终局压缩、最终长期优化相关内容
  - 引力矩阵层的原版配方增强
  - 终局强化、长期 build、全局优化相关内容
- 同阶段附加：
  - 黑雾支线最终批次内容在此阶段收束

### 下一步草案：精馏塔完全体定位

#### 2.3 版本中的最终定位

精馏塔在 2.3 的完整职责直接固定为：

- 只处理矩阵
- 主输出稳定为残片
- 不再直接产奖券
- 不承担核心/芯片/精华等杂项 jackpot 职责

也就是说：

精馏塔 = 常规模式中，将当前阶段矩阵压缩为残片的经济建筑

补充定位：

- 不同矩阵层次对应不同残片效率
- 高阶矩阵应明显优于低阶矩阵
- 黑雾矩阵不并入主线开线池，而作为独立战斗支线输入
- 黑雾矩阵进入精馏塔后，除残片外还可承担黑雾支线推进接口

#### 精馏塔的输入规则

- 优先允许输入当前最高已解锁矩阵层次
- 是否允许输入更低层矩阵，后续再决定
- 若允许低层矩阵输入，则转换效率显著低于当前阶段矩阵
- 黑雾矩阵作为独立战斗支线输入，单独处理

#### 精馏塔的输出规则

- 主输出固定为残片
- 不额外产出新的外部货币
- 不引入额外专属道具
- 不做大杂烩 jackpot 掉落
- 若存在副产物，也只允许服务于现有系统内部进度或黑雾支线推进

#### 精馏塔的成长方向

精馏塔升级只影响以下四类内容：

1. 吞吐量
2. 能耗
3. 残片转化效率
4. 不同矩阵层次之间的转化边际收益差

不影响：

- 是否直接抽到大奖
- 是否直接跳过整条成长链

#### 精馏塔矩阵 -> 残片倍率草案

设计原则：

- 高阶矩阵的残片效率必须显著高于低阶矩阵
- 但不能高到让“把主线矩阵全部拿去精馏”成为唯一最优解
- 低阶矩阵在高阶段可继续精馏，但效率要明显衰减
- 黑雾矩阵作为战斗支线输入，价值高于信息矩阵、低于宇宙矩阵

基础倍率（不考虑阶段衰减）：

| 输入矩阵 | 基础残片 |
|---|---:|
| 电磁矩阵 | 2 |
| 能量矩阵 | 4 |
| 结构矩阵 | 8 |
| 信息矩阵 | 10 |
| 引力矩阵 | 16 |
| 宇宙矩阵 | 32 |
| 黑雾矩阵 | 20 |

阶段衰减系数（相对于当前最高已解锁主线矩阵层）：

| 与当前最高阶段差距 | 系数 |
|---|---:|
| 当前阶段 | 100% |
| 落后 1 层 | 70% |
| 落后 2 层 | 45% |
| 落后 3 层及以上 | 25% |

黑雾矩阵处理建议：

- 不参与主线阶段衰减表
- 只要黑雾支线开放，即按固定倍率处理
- 后续若黑雾支线有更深层成长，可通过精馏塔等级或黑雾里程碑提高其固定倍率

示例：

- 当前最高为结构矩阵时：
  - 电磁矩阵 -> 2 * 45% ≈ 1
  - 能量矩阵 -> 4 * 70% ≈ 3
  - 结构矩阵 -> 8

- 当前最高为引力矩阵时：
  - 电磁矩阵 -> 2 * 25% ≈ 1
  - 能量矩阵 -> 4 * 25% = 1
  - 结构矩阵 -> 8 * 45% ≈ 4
  - 信息矩阵 -> 10 * 70% = 7
  - 引力矩阵 -> 16

倍率意图：

- 电磁 / 能量矩阵可以精馏，但不会因为前期大规模量产而长期统治经济
- 结构 / 信息矩阵开始成为真正有意义的中期残片来源
- 引力 / 宇宙矩阵是后期主动送去精馏才“值”的资源
- 黑雾矩阵作为支线稀有输入，始终有独立价值

#### 精馏塔与模式的关系

- 常规模式：
  精馏塔是稳定层核心建筑

- 速通模式：
  精馏塔存在感弱化，不作为主要资源来源

### 下一步草案：成长池内容表

#### 成长池的顶层定位

成长池是纯确定性系统，不承担随机抽卡职责。

它只负责四类内容：

1. 建筑关键节点突破
2. 阶段跃迁入口
3. 原版配方增强
4. 全星区增幅入口 / 展示

#### 成长池子模块 1：建筑关键节点突破

适用对象：

- 交互塔
- 矿物复制塔
- 点数聚集塔
- 转化塔
- 精馏塔
- 行星内物流交互站 / 星际物流交互站

规则：

- 普通等级：由塔种经验自然积累推进
- 关键节点：使用残片 + 当前阶段矩阵突破
- 推荐关键节点：3 / 6 / 9 / 12

节点语义：

- +3：基础效率档位
- +6：特质 1 解锁
- +9：中后期效率档位
- +12：特质 2 解锁

#### 成长池子模块 2：阶段跃迁入口

定位：
提供跨时代基础设施的定向入口，不做随机。

候选目标：

- 大型采矿机
- 位面熔炉
- 量子化工厂
- 重组式制造台
- 自演化研究站
- 物流站 -> 星际物流站 -> 轨道采集器

明确不放：

- 发电建筑本体
- 黑雾战斗奖励
- 需要复杂燃料链支撑的完整发电体系

消耗结构：

- 当前阶段矩阵为主
- 残片为辅

#### 成长池子模块 3：原版配方增强

定位：
提供“落后一层”的原版配方半定向增强。

开放规则：

- 必须等到下一层科技全部研究完成后，才开放低一层增强
- 不是“解锁下一层科技”就开放

推荐交互：

- 每个已开放增强层，给出 3 个候选增强项
- 玩家从中选 1 个
- 刷新候选消耗残片
- 真正增强消耗残片 + 当前阶段矩阵

增强方向：

- 降低原料消耗
- 缩短原版配方耗时
- 不做过度复杂的多分支词条系统

#### 成长池子模块 4：全星区增幅

定位：
全局被动增幅的统一承载与展示入口。

发放原则：

- 只通过成就系统发放
- 不再通过图鉴页面、任务页面、其他独立系统直接挂 buff

迁移来源：

- 图鉴收集度阈值加成
- 工厂耗电相关加成
- 分馏成功率/损毁/翻倍相关加成
- 物流与上传下载效率相关加成
- 发电阶段增幅

实现目标：

- 成就系统统一汇总最终总加成
- 图鉴 / 任务 / 里程碑只负责触发成就条件

#### 图鉴加成迁移方案

当前图鉴加成：

- >=25% 成功 +2%
- >=50% 成功 +5%
- >=75% 损毁 -1%
- >=100% 翻倍 +3%

后续迁移方式：

- “达到 25% 图鉴收集度” -> 一个成就
- “达到 50% 图鉴收集度” -> 一个成就
- “达到 75% 图鉴收集度” -> 一个成就
- “达到 100% 图鉴收集度” -> 一个成就

由这些成就统一提供增幅。

#### 发电相关增幅迁移方案

发电相关不进入抽奖池，而统一走成就 / 里程碑系统。

推荐类型：

- 工厂电力需求降低
- 当前发电阶段效率提升
  - 风电阶段
  - 火电阶段
  - 聚变阶段
  - 裂变阶段
- 电力相关威胁增长降低（若后续需要）

#### 成长池资源总表

- 建筑关键节点：
  残片 + 当前阶段矩阵

- 阶段跃迁：
  当前阶段矩阵为主，残片为辅

- 原版配方增强：
  残片 + 当前阶段矩阵

- 全星区增幅：
  不消耗，通过成就系统解锁

### 下一步草案：成就系统重构规划表

#### 成就系统的新职责

成就系统在 2.3 之后承担两类职责：

1. 展示与汇总全部全局增幅
2. 提供少量关键的阶段性奖励或启动资源

也就是说：

- 绝大多数“被动加成”都汇总进成就系统
- 主线任务、循环任务、图鉴不再直接挂长期 buff

#### 旧奖励迁移原则

1. 奖券奖励
   - 全部移除
   - 改成当前阶段矩阵、残片、或直接抽取机会

2. 增幅芯片奖励
   - 全部移除
   - 改成：
     - 残片
     - 当前阶段矩阵
     - 或建筑关键节点直接解锁资格

3. 配方核心奖励
   - 重新评估后保留一部分
   - 若原版配方增强与成长池成型，核心奖励应减少

4. 全局 buff
   - 统一保留到成就系统

#### 图鉴迁移到成就的规则

当前图鉴阈值：

- 25%
- 50%
- 75%
- 100%

后续改法：

- “达到 25% 图鉴收集度” -> 成就
- “达到 50% 图鉴收集度” -> 成就
- “达到 75% 图鉴收集度” -> 成就
- “达到 100% 图鉴收集度” -> 成就

图鉴页面只负责显示完成度，不再直接结算增幅。

#### 建议保留的成就类型

1. 分馏总量里程碑
   用于授予少量残片、阶段矩阵、以及部分全局加成。

2. 抽卡 / 开线里程碑
   用于授予开线相关资源，而不是单纯赌池货币。

3. 建筑成长里程碑
   用于授予建筑成长相关的全局被动或残片补差。

4. 配方收集 / 配方满级里程碑
   用于授予原版配方增强、回响推进、全局分馏加成。

5. 符文里程碑
   用于授予符文相关资源和 build 深化奖励。

6. 综合发展里程碑
   用于授予少量高价值全局增幅。

#### 建议从图鉴/任务迁入成就系统的全局增幅类型

1. 分馏成功率
2. 损毁减免
3. 翻倍概率
4. 工厂电力需求降低
5. 当前发电阶段效率提升
6. 上传/下载效率提升
7. 物流效率提升
8. 某类配方整体加成

#### 成就系统最终展示建议

成就面板最终应统一展示：

- 已解锁成就数
- 未解锁隐藏成就数
- 全局总加成汇总
- 分类总加成汇总（可选）

建议不要只显示“成功率 +x%”，而应扩展为：

- 分馏成功率 +x%
- 分馏损毁率 -x%
- 产物翻倍率 +x%
- 工厂耗电 -x%
- 物流效率 +x%
- 当前发电阶段效率 +x%

#### 主线 / 循环任务在新结构中的定位

- 主线任务：
  负责前期引导、首批启动资源、关键系统第一次解锁

- 循环任务：
  负责中后期小额残片、矩阵、抽取机会的稳定补充

- 成就系统：
  负责长期里程碑与全局被动

最终分工：

- 任务负责“推着走”
- 成就负责“永久强”

### 下一步草案：任务 / 成就 / 图鉴重组分工表

#### 总分工原则

- 主线任务：
  前期引导、首次解锁、首批资源

- 循环任务：
  中后期稳定补充，小额资源与少量抽取机会

- 成就系统：
  长期里程碑、全局被动、少量高价值节点奖励

- 图鉴系统：
  只负责展示与统计，不再直接发放增幅

#### 主线任务保留的类型

1. 解锁关键系统
   - 分馏数据中心
   - 第一座关键塔
   - 原胚闭环
   - 精馏塔

2. 完成第一次关键行为
   - 第一次抽卡
   - 第一次开新线
   - 第一次原胚循环
   - 第一次建筑关键节点突破

3. 阶段推进节点
   - 到达某一矩阵层次
   - 开启某个新阶段池内容

主线任务不再承担：

- 长期全局 buff
- 大量重复性资源投喂
- 芯片/奖券发放

#### 循环任务保留的类型

1. 分馏总量类
2. 原胚闭环类
3. 流派聚焦类
4. 工艺精进类
5. 配方收集类

循环任务奖励建议：

- 残片
- 当前阶段矩阵
- 少量直接抽取机会

循环任务不再奖励：

- 实体奖券
- 增幅芯片
- 长期永久 buff

#### 成就系统保留的类型

1. 分馏总量里程碑
2. 抽卡 / 开线里程碑
3. 建筑成长里程碑
4. 配方收集 / 配方满级里程碑
5. 综合发展里程碑
6. 图鉴收集度里程碑
7. 发电阶段 / 工厂阶段 / 物流阶段里程碑
8. 精馏经济 / 阶段跃迁里程碑

#### 现有奖励的迁移规则

##### 直接删除

- 所有奖券奖励
- 所有增幅芯片奖励

##### 保留但重做

- 配方核心奖励
  只保留少量关键节点，不再高频发放

- 残片奖励
  保留，作为最主要的通用成长资源

##### 迁入成就系统

- 图鉴加成
- 发电阶段增幅
- 工厂耗电相关增幅
- 物流效率相关增幅
- 分馏成功 / 损毁 / 翻倍相关增幅

#### 图鉴系统重组

图鉴页面最终只负责：

- 展示各矩阵层的配方解锁情况
- 展示完全升级 / 已解锁 / 总数
- 展示当前已达成的图鉴成就节点

图鉴页面不再直接：

- 计算并发放增幅
- 结算全局 buff

#### 建议删除或重做的现有任务/成就奖励类型

1. “奖励-奖券”
   - 全部删除

2. “奖励-增幅芯片”
   - 全部删除

3. “奖励-原胚”
   - 主线前期少量保留
   - 后续不再大量发放

4. “奖励-配方核心”
   - 从高频奖励改为关键节点奖励

#### 新结构下的奖励优先级

1. 主线任务
   - 首次启动资源
   - 少量矩阵 / 残片
   - 少量原胚

2. 循环任务
   - 残片
   - 当前阶段矩阵
   - 少量抽取机会

3. 成就系统
   - 永久 buff
   - 少量高价值残片 / 矩阵
   - 极少量关键成长项

#### 已确认补充结论

- 现有主线任务仅保留骨架，奖励与语义按迁移表整体重写。
- “幸运抽奖”不保留为独立旧语义任务，统一改写为“开线推进/阶段推进”。
- 原版配方增强相关成就挂在“配方收集 / 配方满级”主类下，不再单独拆一组。
