# 【分馏原胚、分馏原胚（破损）】

分馏原胚（破损）可以转化为分馏原胚（破损）、分馏原胚。

分馏原胚可以转化为分馏原胚（破损）、分馏原胚、各个分馏塔。

这两个物品输入交互塔会转换为残片。

# 【物品交互与交互塔】

## 功能

mod货币数据的来源。可以直接移动背包内的物品（类似机甲装饰），也可以将物品输入交互塔（便于产线输入）。

转换为mod数据的时候，会扣除物品的所有增产点数。

黄糖之后可以选择任意矩阵作为耗材，此时交互塔正面可以输出“分馏原胚”。
开局给予3个矿物复制塔和1个转化塔，并提供蓝糖兑换原胚的功能。前期主要依靠原矿复制得到破损原胚。

# 【原矿复制配方（1A -> 1.1A）与矿物复制塔】

## 功能

复制普通矿物、珍奇矿物、黑雾特有资源。复制时有一定概率产出“分馏原胚（破损）”、“分馏原胚”。

1. 资源越稀有，成功率就越低。具体数值由原版游戏的矿物总数确定。大体上从高到低为：

- 单极磁石（仅黑洞中子星）
- 刺笋 / 光栅（有星球大量存在）
- 分型硅 / 金伯利 / 有机晶体（分布散）
- 可燃冰（巨星） 硫酸
- 黑雾掉落
- 钛石、硅石
- 铁矿、铜矿、煤矿、石矿、油井

2. 矿物开采科技的加成对成功率也生效。

## 建筑

### 矿物复制塔

增产剂可以提升成功率。
分馏塔模型，将物品输入矿物复制塔，输出的物品为0增产点数

### 矿物复制行星接收站

增产剂可以提升成功率和产能。
小塔模型（3槽），每个槽位都可以选一种“原矿”，内部模拟复制直接添加到槽位中

## 配方

### 解锁

无条件。

### 升级

成功产出指定数目的产物后，自动升级。

升级之后，概率提升。

### 突破

升至5级后可以突破，突破需要抽卡得到的配方回响。
星级越高突破需要的回响个数越多，成功率也越低。

突破成功时星级+1，失败时星级不变。成功后，基础概率必定提升，输出堆叠必定+1。
除此之外，还可以选择一项加成，可选的有：
- 概率输出翻倍
- 概率不消耗材料

# 【转化配方（1A -> XA + YB + ZC）与转化塔】

## 功能

将任意物品转换为其他物品。失败时原材料消失，成功时对所有转化可能得到的产物都进行对应的概率判定。
如果判定通过，再看产物数目是否为小数，小数部分再进行概率判定。

随着矩阵研究以及转化次数的增加，基础成功率会逐渐升高。

以1级传送带为例，假设它的转化产物列表为：
- 50% 1级传送带 x 1
- 20% 2级传送带 x 1
- 10% 3级传送带 x 1
- 5% 2级传送带 x 2.5
- 2.5% 2级传送带 x 2.5

那么当成功率5%，损毁率1%，转化100个物品的情况下，5个物品会成功转化，1个损毁，94个流出。

假如某个转化成功的物品判定“5% 2级传送带 x 2.5”通过，则50%概率2个2级带，50%概率3个2级带。

最终期望可以为正数也可以为负数，负数期望可以通过配方突破变为正数。

## 建筑

### 转化塔

增产剂可以提升原始成功率（不影响转化产物列表的成功率）。

## 配方

### 解锁

通过“转化卡池”抽取。抽取需要“转化抽奖券”（由矩阵兑换），越高级的

### 升级

成功转化指定次数后，自动升级。

升级之后，原始成功率提升。

### 突破

升至5级后可以突破，突破需要抽卡得到的配方回响。
星级越高突破需要的回响个数越多，成功率也越低。

突破成功时星级+1，失败时星级不变。成功后，基础概率必定提升，输出堆叠必定+1。
除此之外，还可以选择一项加成，可选的有：
- 概率输出翻倍
- 概率不消耗材料
- 专精转化产物的某一项（该项成功率大幅提升，数目小幅提升，其他项目概率略微下降）

# 【点数聚集（XA，平均4点 -> XA（平均n点（n>4）） + YA（平均低于4点））与聚集塔】

## 功能

将物品的增产点数集中到少部分物品上。原版三种增产剂对应1、2、4点增产点数，而该建筑可以至多聚集至10点。

点数上限与转化成功率如下：
- 初始：上限5点（成功率5%）（上限也可以是4点，但是早于增产剂MK3解锁）
- 红糖：6点（6%）
- 黄糖：7点（8%）
- 紫糖：8点（11%）
- 绿糖：9点（15%）
- 白糖：10点（20%）

成功率受如下加成：
- 矩阵解锁情况（就是上面的5%、6%……）
- 奖池获取的成功率增幅累计

## 建筑

### 聚集塔

点数越多，聚集效率越高。4点时，达到预设的概率。

## 配方

无配方。

# 万物复制配方（1A -> 1.1A）与量子复制塔、宇宙复制塔

## 功能

类似矿物复制，但可以复制所有物品。需要增产点数。

概率与增产点数成正比。输入的物品平均增产点数为10点时，概率为配方设定概率值。平均0点时，概率为0。

物品价值越高，能抽到对应复制配方的概率越低，且对应配方的成功率也越低。

全局黑雾威胁等级越高，概率也就越高。

## 建筑

### 量子复制塔

分馏塔模型。复制时会产出“分馏精华”，可用于直接兑换复制配方。

### 戴森球能量复制锅

射线接收器模型。耗电极高（以此软限定能建造的数目），无杂质，速率极快。选定物品后直接生成。
需要消耗戴森球能量。

## 配方

### 解锁

“复制卡池”抽取。

### 升级

基础速率小幅提升

### 突破

与别的类似，也是5级突破。
每次突破都会重置升级，并使基础概率大幅提升，产物数目小幅提升。

# 【配方解锁方式细节】

## 输入指定量的目标产物

原矿复制配方属于该种方式。向交互塔输入对应物品将有概率解锁。

可以在商店页面浏览哪些配方可以解锁，并查看解锁进度。

每个配方的解锁进度都有最大值，它表示你最多需要多少个对应物品才能解锁该配方。

解锁配方需要的物品数目依据价值而定，价值越高，需要的物品数目会略微降低（但整体价值更高）。

解锁进度预计呈正态分布。
假设上限为1000，则大概率在输入500个物品的时候解锁，至多需要1000个物品解锁。

## 无条件解锁，但普通卡池抽奖来升级

所有转化配方属于该种方式。

正是由于所有转换配方都已解锁，所以在显示可获取的列表中，一开始都是隐藏的。

直至转化出指定量的物品之后，才可以看到物品名称；在转化出更多物品之后，才可以看到概率。

每次转换都会为为配方带来一定经验，经验够了即可升级。

如果想快速升级，就去普通卡池抽奖。

普通卡池的货币为各种原材料分解得到的碎片，抽奖可能获取任意配方的经验。

## 无条件解锁，但特殊卡池抽奖来升级

显而易见，即使有了超越科技的材料，也没办法利用它。但是建筑不受影响。

# 【配方升级与突破】

升级暂定5级为上限，达到上限可以突破。

升级有少量提升，主要体现在概率方面

突破可以定向选择配方修改方向，大幅增强配方的效果

每突破一次，升级都会重置，配方的品质会升高。

白绿蓝紫红金 表示配方品质

# 【商店划分与解锁】

按照糖的类型，分为6层商店（蓝糖商店-白糖商店）。向交互塔输入一定量矩阵即可解锁对应商店。

除此之外，还有黑雾商店，开局解锁，但里面的可兑换物品是随黑雾等级逐步解锁的。
且兑换这里面的材料需要输入一定量黑雾特有产物。

# 【抽卡机制】

抽奖可以获取配方经验、残片（就是狗粮，也可以用于抽普池）、结晶（小概率）。
结晶可以直接兑换想要的配方或对配方进行强化。

抽卡的货币是各个矩阵（对应特殊卡池），或者残片（对应普池）

抽卡无保底机制，概率固定为0.6%。
这是因为对于自动化游戏，产出的物品会越来越多，且大量的物品会直接冲淡概率的波动。

抽卡有喜好机制，可以选择1个非常想要的和3个想要的东西。
如果抽到配方，会有30%为非常想要，各10%为想要，40%其他。
如果连续4次没有获取到非常想要的物品，第5次必定获取。

配方升级只需要使用次数，但突破需要大量残片，甚至会需要结晶。

或者这样设计：
抽奖可以获取配方，已有配方配方会转换为同名结晶，配方突破需要结晶。
突破到上限后，自动分解为结晶碎片，用于兑换通用结晶，来换取其他结晶。

附原神抽卡机制：
在角色池中，1至73抽时，每次抽卡的概率为0.6%。
从74抽开始，每抽的概率逐渐递增约6%，直到90抽时达到100%。
具体来说，出金时有50%的概率会出当期up角色，50%的概率出常驻角色。
出金概率在74至90抽之间逐步提升，平均up角色的期望抽数为93.36抽。
